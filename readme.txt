== Description ==

This is a plugin that enables a custom gutenberg block to fetch info from the fnugg API. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/jpg-fnugg-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
